Memory infrastructure for agents and teams.

Engram Cloud turns the Engram engine into a hosted service with workspaces, API keys, an MCP-style bridge, starter skills, and a dashboard teams can actually use.

Recommended hosting: VPS. Engram works best as a warm Python service with direct Postgres access. Vercel is fine for a thin shell, but it is the wrong place to run the memory runtime itself.

Built for a long-lived runtime
